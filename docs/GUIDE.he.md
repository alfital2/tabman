# Tabman — מה חדש ואיך משתמשים

## ⌨️ קיצורי קצב
בוחרים תו (או גוררים בחירה) ולוחצים:

| מקש | ערך |
|---|---|
| `q` | שלם |
| `w` | חצי |
| `e` | רבע |
| `r` | שמינית |
| `d` | שש־עשרית |
| `f` | 32 |
| `.` | נקודה (מחזוריות: · ·· בלי) |

המקש גם קובע את ערך ההקלדה הבא. רוצים מקשים אחרים? בחלון הקיצורים (כפתור ⌨ למעלה) לוחצים על מקש → מקלידים חדש. `⌫` מבטל, יש Reset.
שימו לב: Let ring עבר מ־`r` ל־`g`.

## 🎵 טריולות וחלוקות
תו מסומן + `Ctrl+3` → טריולה. עובד עם 2–9 (`Ctrl+5` = קווינטולה וכו').
`Ctrl+1` מבטל. יש גם כפתורים בפאנל הימני (TUPLETS).

## 🖱️ בחירה של כמה תיבות
גרירה = בחירה. **Shift+גרירה = מוסיף לבחירה** (כמו קבצים בשולחן העבודה). Shift+קליק על תו בודד מוסיף/מוריד אותו.
`⌘C` על בחירה שחוצה תיבות שומר את מבנה התיבות; `⌘V` מדביק תיבה־תיבה, כולל משקלים.

## 🗑️ מחיקה
`⌫` ראשון → התווים הופכים להפסקות. `⌫` שני על אותה בחירה → ההפסקות נעלמות והתיבה מתהדקת.

## 🎼 תיבת פתיחה (קדמה)
קליק ימני על התיבה הראשונה → **Pickup bar (anacrusis)**.
התיבה מתנגנת רק באורך התוכן שלה, בלי מספר, והמספור מתחיל מהתיבה הבאה.

## 🔁 חזרות וסיומות
קליק ימני על תיבה → שורת **Repeats & endings**:

| צ'יפ | פעולה |
|---|---|
| 𝄆 | תחילת חזרה |
| 𝄇×2 ×3 ×4 | סוף חזרה + כמה פעמים |
| 1. / 2. / 1.2. | סיומת ראשונה / שנייה / משותפת |
| ✕ | ניקוי הכל מהתיבה |

הנגינה (Space) מתנגנת עם החזרות והסיומות באמת — כולל המטרונום, והסמן קופץ יחד איתן.

הכל ניתן לביטול ב־`⌘Z`, והכל נשמר בקובץ.
